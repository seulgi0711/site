## HTML 표현 방식

### React

JSX를 이용해서 유사 HTML을 표현한다.

camel-case를 사용해서 attribute및 props를 표현한다.

```tsx
<input onChange={handleChange} value={value} />
```

### Svelte

HTML과 유사한 형태로 HTML을 표현한다. 말그대로 유사한것이기 때문에 컴파일러를 통해 컴파일 해야 한다.

```js
<input on:change={handleChange} value={value} />
```
